# MERN Blog App

## Overview
This is a simple blog application built using the MERN stack (MongoDB, Express, React, Node.js).  
It allows users to perform CRUD operations on blog posts.

---

## Features
- Create a blog post
- View all posts
- Update a post
- Delete a post

---

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
